ゲームタイトル: 物理障壁に勝てないおばけ

ゲーム概要

  • 開発人数: 計19人
  • コンセプト: おばけらしくないおばけでパズルを解く
  • 開発期間: 2023年9月 ～ 2024年2月
    • プランナー2人
    • プログラマー4人（＋メンター2人）
    • 3Dモデラー8人
    • エフェクトデザイナー1人
    • 2Dデザイナー1人
    • PV担当1人

アピールポイント

  • ビジュアル面: トゥーン調で統一し、かわいいアニメっぽさを演出
  • ゲーム性: スキルを駆使してクリア方法を考える楽しさ

担当箇所

  • リードプログラマー: プログラミングの相談、仕事の割り振り、モデルやアセットの導入
  • プログラマー:
    • コントローラーの入力管理
    • プレイヤーの移動
    • スキル3種
    • ステージオブジェクトのギミック
    • タイマーやリザルトのユーザーインターフェース

主要言語・ツール

  • 主要言語: C#
  • 主要ツール: Unity, GitHub, Microsoft Visual Studio 2022, UniTask, DOTween

コード解説

  • コントローラーの入力管理: InputSystemを用いてコントローラーからの入力を管理し、シングルトンパターンとDontDestroyOnLoadを使用。L1およびR1ボタンでカメラ回転。

    ControllerManager.cs

  • プレイヤーの動作制御: UniTaskを使用した非同期処理で移動とスキル管理。コントローラー入力を適切に処理。

    PlayerController.cs

  • プレイヤーの移動管理: UniTaskを用いた非同期アニメーションで移動。移動先のチェックや障害物の確認等。

    PlayerMove.cs


    PlayerRotate.cs


    ObjectRotation.cs

  • スキル管理: スキル発動と管理。範囲表示やフェード効果、発動位置の確認。

    SkillManager.cs


    SkillAreaDisplay.cs


    AreaColorFade.cs

  • スキル: スキル３種の処理　オブジェクトの破壊、オブジェクトの移動、オブジェクトの非表示、再配置等

    Skill.cs


    SkillFire.cs


    SkillWater.cs


    SkillSuction.cs

ゲームタイトル: カジキvsクトゥルフ

ゲーム概要

  • コンセプト: B級映画のようなシュールさを楽しむシューティングゲーム
  • 開発期間: 2023年4月 ～ 2023年9月
  • 開発人数: 計22人
    • プランナー5人
    • プログラマー5人（＋メンター3人）
    • 3Dモデラー4人
    • エフェクトデザイナー2人
    • 2Dデザイナー2人
    • PV担当1人

アピールポイント

  • ビジュアル面: サバとカジキのモデリングにこだわり、それらが撃った後も残ることでシュールな演出を実現
  • ゲーム性: クトゥルフの圧倒的な力を感じつつも、B級映画のようなシュールさを楽しめるゲーム体験

担当箇所

  • 開発スケジュール管理とモデルの導入確認: スケジュールの調整やモデルの適切な導入を担当
  • プログラム:
    • 遷移管理
    • カメラアニメーションの実装

主要言語・ツール

  • 主要言語: C#
  • 主要ツール: Unity, GitHub, Microsoft Visual Studio 2022, UniTask, Chinemachine

コード解説

  • 遷移管理: シーン遷移の呼び出しを簡略化し汎用性を持たせることに注力。フェード処理にUniTaskを使用。

    SceneCOntroller.cs


    FadeManager.cs

  • カメラアニメーション: Chinemachineを使用してゲームシーンとボス戦シーンのカメラアニメーションをコードレスで実装。
